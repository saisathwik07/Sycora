## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SaiSathwikReddyN/TaskManagementApp.git
cd TaskManagementApp
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
npm install
ng serve
```
